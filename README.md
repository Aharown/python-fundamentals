# python-fundamentals

Working through [CS50P — Introduction to Programming with Python](https://cs50.harvard.edu/python), Harvard University's free Python course. Each problem set is solved locally in VS Code and committed after each session.

This repo is part of a structured self-directed engineering study programme alongside DSA practice and independent project work.

---

## Structure

```
python-fundamentals/
├── pset0/    # Functions and Variables
├── pset1/    # Conditionals
├── pset2/    # Loops
├── pset3/    # Exceptions
├── pset4/    # Libraries
├── pset5/    # Unit Tests
├── pset6/    # File I/O
├── pset7/    # Regular Expressions
└── pset8/    # Object-Oriented Programming
```

---

## Approach

Each problem set is completed locally before being pushed. Solutions are written to pass the problem requirements and to be readable — not just correct. Where there are multiple valid approaches, comments in the file explain the reasoning behind the one chosen.
